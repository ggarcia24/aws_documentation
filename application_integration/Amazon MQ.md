# Amazon MQ

## Features

- Managed Apache ActiveMQ
- Doesn't automatically scale
- Runs on a dedicated machines
- Can run in HA with failover
- Supported Protocols
    - OpenWire
    - JMS
    - NMS
    - AMQP
    - MQTT
    - Stomp
    - WebSocket
- Supports Queues and Topics
